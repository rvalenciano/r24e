API Endpoints

Requires authentication!


Empresa

    Parámetros:
  • Id: Integer (Opcional), id de la empresa autobusera.

  • Nombre : String (Opcional), nombre de la empresa autobusera.

  • Si no se proporciona parámetros, devuelve todas las empresas.

Rutas

    Parámetros:
  • Id: Integer (Opcional), id de la ruta.

  • NombreRuta : String (Opcional), nombre de la ruta.

  • Si no se proporciona parámetros, devuelve todas las rutas.

Paradas

    Parámetros:
  • lat: Float (Requerido), Latitud del punto a buscarle una parada cercana.

  • long: Float (Requerido), Longitud del punto a buscarle una parada cercana.

  • Si no se proporciona parámetros, devuelve todas las paradas.

Segmento

    Parámetros:
  • lat: Float (Requerido), Latitud del punto a buscarle un segmento cercano.

  • long: Float (Requerido), Longitud del punto a buscarle un segmento cercano.

  • Si no se proporciona parámetros, devuelve todos los segmentos.
